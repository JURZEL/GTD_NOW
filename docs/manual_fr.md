# GTD Student — Manuel utilisateur (Français)

Ce manuel explique les fonctionnalités de l'application et comment les utiliser.

## Vue d'ensemble

GTD Student est une petite application Getting Things Done pour les étudiants. Elle aide à capturer, organiser et revoir les tâches et projets.

## 1 — Saisir

- Saisie rapide: utilisez le FAB pour ajouter rapidement une tâche ou une idée à la boîte de réception.
- Ajoutez un titre et, si besoin, une description, un contexte, une estimation de temps et un niveau d'énergie.

## 2 — Boîte de réception

- La boîte de réception est l'espace de préparation. Traitez les éléments régulièrement.
- Pour chaque élément, vous pouvez:
  - supprimer
  - marquer comme fait
  - le transformer en tâche avec date d'échéance
  - le déplacer vers un projet
  - le différer vers "Peut-être/Plus tard"

## 3 — Projets

- Un projet regroupe des tâches liées pour atteindre un résultat.
- Créez un projet, ajoutez des tâches et ouvrez-le pour voir les tâches associées.

## 4 — Actions (Next Actions)

- La vue Actions liste les tâches que vous pouvez faire ensuite, filtrables par contexte, niveau d'énergie et fenêtre temporelle.
- Utilisez les filtres pour trouver des tâches adaptées à votre situation.

## 5 — Rappels & Dates d'échéance

- Activez les notifications dans les paramètres pour recevoir des rappels.
- Sur les plateformes sans notifications natives, des rappels de secours sont disponibles (voir Paramètres).

## 6 — Revue

- Effectuez une revue hebdomadaire pour maintenir les projets à jour et planifier la semaine suivante.

## Avancé

- Export/Import: sauvegardez ou restaurez vos données via la section Backup.
- Paramètres: modifiez le thème, la langue, les présélections de snooze et les préférences de notification.

## Confidentialité & Données

- Les données sont stockées localement dans Hive. Aucune synchronisation distante n'est activée par défaut.
